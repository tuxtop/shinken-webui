# Shinken WebUI

Alternative Web User Interface for Shinken

## Requirements

### Hardware

- Linux OS (Debian, Ubuntu, RedHat, CentOS, etc.)
- Can work on a simple VM w/
 - 1 vCPU
 - 2 Gb RAM
 - 25 Gb Disk

### Software

- Shinken
- mod_livestatus enabled
- PHP 5.6
- [PHP PECL YAML](http://php.net/manual/fr/book.yaml.php)
